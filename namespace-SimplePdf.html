Namespace SimplePdf

Classes summary

Page Extension of ZendPdf\Page which allows using arbitrary units (e.g. inches or centimeters) and works from top to bottom, instead of default PDF/PostScript geometry. It also adds some basic formatting functionality, like word wrap, margins and text alignment.
